Project Overview

A university team project where we had Computer Science and Cyber Security students work together to complete a game, website, game art, game design documents, security tests, etc. Computer Science and Cyber Security students worked on delivering an 'Online Multiplayer Turn-based game which makes use of a database in some way'. This project showcases the main deliverable, the game. I was responsible for the development, where I took on the role of checking over peers code, peer coding, etc.

Technologies Used

Unity
Photon PUN 2
MySQL

Key Features

Turn Based Mechanics (Multiplayer)
User Profiles, keep record of wins/losses (Kills, death and KD ratio) (MySQL)
Health Bar
Fall Damage
Projeciles (Shuriken, katana, ...)
Projectile Damage

What's next?

I would like to revisit this game as I believe there is a potential interest in this game genre. There are a few things I would add or improve to make the game even better...
1. Animations
With my newly attained skills in 3D modelling and animation, I would like to reivist this project in the future and make use of 3D animations, but gameplay would remain on a 2D plane.
2. Character colour selection The code for this already exists, however, we took it out for the 2D animation instead. This will be reintroduced in a later update.

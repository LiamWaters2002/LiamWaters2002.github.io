Project Overview

A university team project where Computer Science and Cyber Security students collaborated to create a game, website, game art, game design documents, security tests, etc. The project involved delivering an 'Online Multiplayer Turn-based game that utilizes a database in some way'. This project primarily showcases the game, and I was responsible for its development, which included code review and pair programming.

Technologies Used

  • Unity
  • Photon PUN 2
  • MySQL

Key Features

  • Turn-Based Mechanics (Multiplayer)
  • User Profiles to keep a record of wins/losses (Kills, deaths, and KD ratio) (MySQL)
  • Health Bar
  • Fall Damage
  • Projectiles (Shuriken, katana, ...)
  • Projectile Damage

What's next?

I would like to revisit this game, as I believe there is potential interest in this game genre. There are a few things I would like to add or improve to make the game even better:

  1. Animations: With my newly acquired skills in 3D modeling and animation, I would like to revisit this project in the future and incorporate 3D animations, while keeping gameplay on a 2D plane.

  2. Character Color Selection: The code for this feature already exists; however, we removed it for the 2D animation. It will be reintroduced in a later update.

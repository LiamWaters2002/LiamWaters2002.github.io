Project Overview

A city-building game designed for mobile devices. The game utilizes functional graphs to display economic outcomes based on economic decisions. In economics, understanding various graphs is essential, and this project aims to provide a visual representation of economic outcomes on dynamic graphs to enhance the user's understanding of economic shifts.

Technologies Used

  • Unity

Key Features

  • Functional Graphs (Supply / Demand Graph)
  • Grid Placement & Rotation System
  • Mobile Controls
  • Real-Time Progression (Year, Month)

What's Next?

This is a very ambitious project of mine, and I have plans to expand its breadth and depth, including:

  1. Agents: Economic agents that roam the city, using dynamic pathfinding algorithms to simulate customers going to work, or buying and selling items in the economy.

  2. More Graphs: Including graphs for displaying Government spending, GDP, etc.

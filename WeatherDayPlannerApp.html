Project Overview

A weather planner app, gives details on upcoming weather based on user's location. Suggests locations nearby by using Google Maps.

Technologies Used

Google Maps API
Google Places API
Visual Crossing Weather API
SQLite
Java
Andriod Studio

Key Features

Android Jetpack Navigation
Persistent storage
Display Locations on an interactive map using Markers
Weather Preference Filtering
List locations based on weather at specific times of the day
Filtering system

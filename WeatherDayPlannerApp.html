Project Overview

A weather planner app that provides details on upcoming weather based on the user's location. It also suggests nearby locations using Google Maps.

Technologies Used

  • Google Maps API
  • Google Places API
  • Visual Crossing Weather API
  • SQLite
  • Java
  • Android Studio

Key Features

  • Android Jetpack Navigation
  • Persistent storage
  • Display Locations on an interactive map using Markers
  • Weather Preference Filtering
  • List locations based on weather at specific times of the day
  • Filtering system

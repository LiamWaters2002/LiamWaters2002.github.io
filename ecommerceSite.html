Project Overview

An e-commerce web application designed for seamless buying and selling experiences. Users can add items to sell, explore products, and make purchases from other users..

Technologies Used

  • HTML
  • CSS
  • Javascript
  • C#
  • ASP.NET
  • React
  • Bootstrap
  • MS SQL Server

Key Features

  • User Login/Registration
  • User Authentication with JWT Web Tokens (Token expiration log-out)
  • Web APIs
  • Displaying User/Item/Order Information in Tables
  • Add/Alter/Delete Data within the Database
  • Calculate Total Cost of Basket
  • Calculate Total Revenue of Orders
  • React Routing
  • Responsive Design

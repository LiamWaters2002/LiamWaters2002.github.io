Project Overview

A tool which runs on desktop. It currently allows you to store and share html files, which you can access by loggining in. There is an admin system so other users can access these files.

The tool has had a different origin to begin with, a family member wanted it for Law Software, where he would provide resources for for different areas of law.

Technologies Used

Java
JavaFX
Maven
MySQL
SQLite
CSS

Key Features

Admin System - Upload html files, directories, users. Remove html files, directories, users.

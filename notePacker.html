Project Overview

A tool that runs on desktop. It currently allows you to store and share HTML files, which you can access by logging in. There is an admin system so other users can access these files.

The tool had a different origin to begin with; a family member wanted it for Law Software, where he would provide resources for different areas of law.

Technologies Used

  • Java
  • JavaFX
  • Maven
  • MySQL
  • SQLite
  • CSS

Key Features

Admin System - Upload HTML files, directories, users. Remove HTML files, directories, users.

What's next?

The next plans for the note packer are to generate flashcards from the HTML files. This would be based on the patterns users use to type notes or making use of the patterns used for the HTML file.

  1. Fetch contents based on patterns
    Users will be able to enter a pattern they used in their note-taking, content will be fetched, then displayed on flashcards.

  2. Paid Notes
    Allow users to buy notes from one another.
